# Project Instructions

## Stack
- React + TypeScript scaffolded with Vite
- Package manager: `pnpm`
- UI framework: `tailwindcss` + `shadcn/ui` (`base` version)

## Codex agent note
- Prefer existing Codex skills for edits when available (for example `build-web-apps:shadcn` for UI/component tasks and `build-web-apps:frontend-testing-debugging` for UI validation).
- Do not implement shadcn components manually when a component is available in the configured registry.
